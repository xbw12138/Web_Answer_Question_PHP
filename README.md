#欢迎使用选题系统

运行环境 Apache+mysql+php5.6

在mysql中新建webwork数据库，
导入根目录下的webwork.sql到webwork数据库

上传WebWork到服务器，修改lib文件夹下的db_config.php
中数据库配置。

本系统分为学生答题系统与老师出题系统
学生系统已经开放注册，老师系统暂时不允许注册，只能管理员后台添加。
